dwm Keybindings

Apps and Scripts

MODKEY + Return: Open terminal
MODKEY + b: Open browser
MODKEY + Shift + B: Open Bluetooth manager
MODKEY + h: Open btop
MODKEY + y: Open yazi
MODKEY + n: Open nvim
MODKEY + p: Open pavucontrol
MODKEY + w: Open network manager
MODKEY + e: Open file manager
MODKEY + c: Open vscode
MODKEY + z: Open zeditor
MODKEY + d: Open dmenu
MODKEY + l: Logout
MODKEY + u: Update arch
MODKEY + v: Open clipboard manager
MODKEY + a: Open man page search
MODKEY + Shift + F: Toggle fullscreen
MODKEY + Shift + S: Take screenshot
MODKEY + Shift + W: Change wallpaper
MODKEY + q: Close window

Volume, Brightness, and Media Controls

XF86AudioMute: Mute audio
XF86AudioMicMute: Mute microphone
XF86AudioLowerVolume: Lower volume
XF86AudioRaiseVolume: Raise volume
XF86MonBrightnessUp: Increase brightness
XF86MonBrightnessDown: Decrease brightness
XF86AudioPlay: Play/Pause media
XF86AudioNext: Next track
XF86AudioPrev: Previous track

MODKEY + Ctrl + Left: Decrease master factor
